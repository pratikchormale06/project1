About Maltego

Maltego is a powerful data mining and intelligence gathering tool that provides a unique perspective on relationships between pieces of information on the internet.

Key Features

  • Visual link analysis
  • Real-time data mining
  • Integrated transforms for data gathering
  • Multiple data source integration
  • Custom transform development
  • Graph-based visualization
  • Collaborative investigation capabilities

Available Editions

Community Edition (CE)

  • Free version
  • Limited transforms
  • Basic features
  • Community support

Classic Edition

  • Professional use
  • Full transform set
  • Advanced features
  • Technical support

XL Edition

  • Enterprise use
  • Extended capabilities
  • Premium support
  • Collaboration features

Installation

Download and Install

  1. Visit the official Maltego website
  2. Choose your edition
  3. Download the appropriate version for your OS
  4. Run the installer
  5. Follow the setup wizard

Tip: Maltego is also pre-installed in Kali Linux under Applications → Information Gathering → Maltego

Basic Usage Guide

1. Starting a New Investigation

  1. Launch Maltego
  2. Create a new graph
  3. Select an entity type from the palette
  4. Drag and drop it onto the graph
  5. Double-click to edit entity details

2. Running Transforms

  1. Right-click on an entity
  2. Select "Run Transform"
  3. Choose appropriate transform
  4. View results on the graph

3. Common Entity Types

  • Domain
  • IP Address
  • Email Address
  • Person
  • Organization
  • Website
  • Social Media Profile
  • Phone Number

Best Practices

  • Data Organization: Use meaningful entity names and descriptions
  • Graph Management: Keep graphs focused and organized

Tips and Tricks

Performance

  • Use bookmarks for common transforms
  • Clear graph cache regularly
  • Limit transform scope when possible
  • Use machine settings for optimal performance

Visualization

  • Use different layouts for better clarity
  • Group related entities
  • Use custom entity colors
  • Export graphs in different formats

Investigation

  • Start with known entities
  • Use multiple data sources
  • Verify findings manually
  • Document your methodology
